## Overview

Digital Exclusion Across the United States is a data analysis and geospatial visualization project that investigates disparities in household internet access across the United States using data from the **American Community Survey (ACS)**. The project identifies regions experiencing higher levels of digital exclusion and presents the results through interactive maps and visualizations.

Using statistical analysis, geographic data processing, and GIS visualization techniques, the project transforms raw census data into accessible visual insights that highlight patterns of internet accessibility at both the state and county levels.

This project provided hands-on experience working with large public datasets, geographic information systems (GIS), interactive mapping, data visualization, and reproducible data analysis as part of the **Data Science for the Public Good (DSPG)** internship.

---

## Project Objective

The objective of this project was to:

- Analyze American Community Survey (ACS) data related to household internet access
- Identify geographic disparities in digital connectivity across the United States
- Process and clean census and geographic datasets
- Develop interactive state and county-level maps
- Create visualizations that communicate patterns of digital exclusion
- Support research through reproducible data analysis and documentation
